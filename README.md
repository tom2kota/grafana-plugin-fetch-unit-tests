# Grafana Panel Plugin - Giphy Images 

Giphy images Fetched from endpoint: https://developers.giphy.com/docs/api/endpoint/

-------------------

## Prerequisites
- Grafana 7.0
- NodeJS 12.x
- yarn or npm

-------------------

In console run:
``` 
    npm i
    npm audit fix
    npm run watch
```

Go to web-browser [localhost](http://localhost:3000/)

-------------------
